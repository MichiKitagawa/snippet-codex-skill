# Snippet Codex Skill

This repository distributes the public Codex Skill for operating Snippet rooms through the browser bridge.

It contains only the Skill package. It does not contain the Snippet application source code.

## Install

Download the latest `snippet-browser-bridge.zip` from Releases and install it as a Codex / OpenAI Skill.

The Skill teaches Codex to use the room memory, canvas, appearance, workbench, commit, share, viewer, and past-commit surfaces exposed in the browser. It is written for the current 2D room canvas and does not assume the older 3D/WebGL editing model.
