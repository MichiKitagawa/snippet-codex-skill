# Snippet Codex Skill

This repository distributes the public Codex Skill for operating Snippet through its browser bridge.

It contains only the Skill package. It does not contain the Snippet application source code.

## Install

Download the latest `snippet-browser-bridge.zip` from Releases and install it as a Codex / OpenAI Skill.

## Contents

- `snippet-browser-bridge/SKILL.md`
- `snippet-browser-bridge/agents/openai.yaml`

## Compatibility

- Skill: `snippet-browser-bridge`
- Snippet bridge: R2 Codex-ready workspace
